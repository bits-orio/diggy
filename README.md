# Diggy

The world is solid rock. Dig it out, support your ceilings, and grow a factory in the caves you carve — collapses bury the careless.

A standalone Factorio 2.0 mod reimagining [RedMew's Diggy scenario](https://github.com/Refactorio/RedMew), with first-class optional [Multi-Team Support](https://mods.factorio.com/mod/multi-team-support) integration: per-team cave worlds, custom milestones, and cross-team collapse alerts. Optional [Open Discord Bridge](https://mods.factorio.com/mod/open-discord-bridge) integration mirrors the drama to Discord.

**Requires a brand-new map.** The world is generated by the mod; it cannot take over an existing save.
